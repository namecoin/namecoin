Autotools build should 'just' check most libs, dependencies 
and tell you if you need something else.

Try this

$git clone git://github.com/randy-waterhouse/Open-Registrar.git
$cd Open-Registrar
$git clean -x -f -d

$autoreconf -i -f

$mkdir build && cd build

$../configure --help

(NB: shows configure options, use them wisely)

$../configure options-blah-blah.. --prefix="somewhere safe"

$make

(supports threaded make so can use -j2, -j4 ... ~1 GB RAM needed per thread)

$make install
